# expo-theme-switcher
This repository is an example to apply themes to React Native projects.

## See the project on a device
I published this project at Expo Dev. Then you can scan the project QR Code to see the result on a device by [Expo Go App](https://expo.dev/client).

![expo-theme-switcher-qr-code](./assets/expo-go.svg)

Or access the [this link](https://expo.dev/@marcelofreires/expo-theme-switcher) on your device.

## Run this project
To run this example you must install the dependencies.
```bash
yarn
```

And finally, to build the project you must run the following commands on your terminal.

To build for iOS.
```bash
yarn ios
```

And to build for Android.
```bash
yarn android
```
